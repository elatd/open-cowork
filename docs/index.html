Tip: we recommend downloading the .dmg file from the latest release.

Install in 3 steps

  1. Download the latest macOS release asset.
  2. Open the .dmg and drag the app into Applications.
  3. Launch the app and follow the setup prompts.

macOS security prompt

If macOS shows “App can’t be opened”, go to System Settings → Privacy & Security and click “Open Anyway”. This is standard for new desktop apps.
