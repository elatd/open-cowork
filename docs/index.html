3. Configure

Add your API keys in .env and start the backend server locally.

macOS install steps

  1. Download the latest DMG from the GitHub releases page .
  2. Open the DMG and drag the app into Applications.
  3. If you see a Gatekeeper warning, open System Settings → Privacy & Security and click Open Anyway.
  4. Start the backend server with cd server && npm start.
  5. Launch the desktop app from Applications.
