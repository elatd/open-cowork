Install on macOS

  1. Download the latest .dmg from the button above.
  2. Open the disk image and drag Open Claude Cowork into Applications.
  3. Launch the app from Applications.
  4. If macOS blocks the app, go to System Settings → Privacy & Security, then click Open Anyway.

What you need

  • API keys

    Get keys for Anthropic, Opencode (optional), and Composio.

  • Internet access

    The backend runs locally but connects to provider APIs.

  • macOS 12+

    Recommended for the best Electron runtime support.
